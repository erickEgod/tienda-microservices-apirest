1) instalamos swagger de nestjs:
	- npm install @nestjs/swagger --SE

2) En el main agregamos dos constantes una para configuraciones y otra para crear el documento swagger:
	- const config = new DocumentBuilder()
    .setTitle('Cats example')
    .setDescription('The cats API description')
    .setVersion('1.0')
    .addTag('cats')
    .build();

	- const document = SwaggerModule.createDocument(app, config);

3) Iniciamos el modulo swagger con:
	- SwaggerModule.setup('api', app, document);


4) Agregamos tags en los controladores con:
	- @ApiTags("users")

5) Agregamos descripciones en los dtos con:
	- @ApiProperty({ example: "16,50", description: "Price of the product" })
